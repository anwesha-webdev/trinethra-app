# 🚀 Text Analyzer

A simple web application that analyzes text and provides useful insights.

## ✨ Features
- Word count
- Character count
- Sentence count
- Longest word detection
- Estimated reading time

## 🛠 Tech Stack
- HTML, CSS, JavaScript (Frontend)
- Node.js, Express (Backend)

## ▶️ How to Run

1. Clone the repo
2. Install dependencies:
   npm install
3. Start server:
   node server.js
4. Open index.html using Live Server
